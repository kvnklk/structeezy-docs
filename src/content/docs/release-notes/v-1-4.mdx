## New Pattern Hero Zero

A new Hero Pattern is now available in Structeezy. This pattern is designed to help you scaffold hero sections for your website. It includes the basic elements for a hero section.

## Other improvements

- Fixes code highligting issues in the editor that were introduced with the latest Codemirror update.
- Fixes styles and improves performance of the admin app loader.
- Fixes some styles conflicts with Bricks defaults in the editor.
